Visual Affect Labeling

This website is used to collect a dataset about emotion within images. For each image, you will be asked to respond about the main character's emotion, the number of people in the image, and if the image is a good image for the purposes of data collection on emotion. If you come across an image that has no people or far too many people to classify a main person, please write "none" in the emotion input box so the image can be removed from this set. Emotion can be described with words or short phrases and multiple descriptors should be separated by commas (, ). You can view the data here. Thank you for your help!
